Funcionalidades

Tudo que você precisa.
Nada que você não precisa.

Projetado para ser simples, rápido e completamente seu.

✍️

Editor híbrido WYSIWYG

Escreva Markdown puro ou use a toolbar. O editor entende os dois — # Título vira H1 automaticamente, **negrito** formata na hora.

# Título da nota
**negrito** e *itálico*
- [ ] tarefa pendente
- [x] tarefa concluída
💾

Auto-save

Salva automaticamente enquanto você digita. Sem botão salvar, sem perder nada.

600ms debounce
📁

Arquivos abertos

Cada nota é um arquivo .md real no seu disco. Abra com qualquer editor, faça backup, use no Git.

Zero lock-in
🔍

Busca full-text

Encontre qualquer coisa em todas as suas notas em tempo real.

#

Tags via hashtag

Use #projeto no texto e a tag aparece automaticamente na sidebar para filtrar.

🎨

Temas e cores

Dark e Light mode com 7 accent colors. Tudo persistido localmente.

📌

Fixar notas

Notas importantes sempre no topo da sidebar.

🕐

Histórico automático

Últimas 10 versões de cada nota salvas automaticamente em /history.

🔗

Links internos

Use [[nome-da-nota]] para navegar entre notas ou criar novas automaticamente.

Como funciona

Simples como deve ser.

01

Baixe o executável

Um único arquivo .exe de ~9MB. Sem instalação, sem dependências, sem admin.

→
02

Execute de qualquer lugar

Coloque num pendrive, numa pasta, onde quiser. O app cria a pasta /notes ao lado do executável.

→
03

Comece a escrever

Crie sua primeira nota em segundos. Tudo salvo automaticamente como arquivos .md.

Estrutura de arquivos em runtime
📁 qualquer-pasta/
  ▶ mddesk.exe          ← só isso para rodar
  📁 notes/
    📄 minha-nota.md
    📄 ideias-de-projeto.md
  ⚙ config.json         ← preferências
  📁 history/             ← snapshots automáticos
