# ◈ MD Desk

> **Portable Markdown Notes. Local-first.**

Editor de notas desktop para Windows com suporte completo a Markdown. Sem instalação, sem cloud, sem cadastro — só você e seus arquivos.

## 🚀 Como usar

1. Baixe o `mddesk.exe`
2. Coloque em qualquer pasta (pendrive, Desktop, onde quiser)
3. Execute — pronto

Na primeira execução, o app cria automaticamente:

```
📁 sua-pasta/
  ▶ mddesk.exe
  📁 notes/        ← suas notas ficam aqui
  ⚙ config.json   ← preferências
  📁 history/      ← histórico automático
```

---

## ✨ Funcionalidades

- **Editor híbrido** — escreva Markdown puro ou use a toolbar (H1–H3, negrito, itálico, código, links, listas, checklist)
- **Auto-save** — salva automaticamente enquanto você digita, sem botão salvar
- **Arquivos abertos** — cada nota é um `.md` real no seu disco, sem lock-in
- **Preview split** — editor e preview lado a lado, ou só um deles
- **Busca full-text** — encontre qualquer coisa em todas as notas em tempo real
- **Tags via hashtag** — use `#projeto` no texto e filtre pela sidebar
- **Fixar notas** — notas importantes sempre no topo
- **Links internos** — `[[nome-da-nota]]` navega ou cria notas
- **Temas** — Dark / Light + 7 accent colors
- **Histórico automático** — últimas 10 versões de cada nota salvas em `/history`
- **100% offline** — zero dependência de internet
